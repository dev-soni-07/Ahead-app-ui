
# Dribble design --> website

This is a dribble design, converted to a Next js 13 app.
## Run Locally

Clone the project

```bash
  git clone https://github.com/Ajaydeep123/Ahead-app-ui.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
